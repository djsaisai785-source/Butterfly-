# Butterfly — Design System

## Brand
- **Name**: BUTTERFLY
- **Tagline**: L'univers de la nuit, en un seul endroit.

## Colors
- `--bg-primary`: #0A0A0F (noir profond)
- `--bg-secondary`: #12121A (noir secondaire)
- `--bg-card`: #1A1A26 (carte)
- `--accent-gold`: #D4AF37 (or classique)
- `--accent-rose-gold`: #B76E79 (or rose)
- `--accent-platinum`: #E5E4E2 (platine)
- `--accent-amber`: #FFBF00 (ambre/bling)
- `--text-primary`: #F5F5F0 (blanc cassé)
- `--text-muted`: #8A8A9A (gris doux)
- `--border`: #2A2A3A

## Typography
- **Display**: Playfair Display (serif, élégant)
- **Body**: Poppins (sans-serif, lisible)
- **Accent**: Cormorant Garamond (serif luxe pour sous-titres)

## Spacing
- Base unit: 4px
- Cards: padding 20px, border-radius 16px
- Sections: padding 48px vertical

## Style
- Fond toujours sombre (noir profond)
- Accents dorés/or rose sur éléments clés
- Glassmorphism subtil sur les cartes (backdrop-blur, border gold 1px)
- Gradients: noir vers violet profond (#1a0a2e) pour les heroes
- Icônes: lucide-react (web), phosphor-react-native (mobile)

## Components
- **Boutons primaires**: fond or (#D4AF37), texte noir, border-radius 8px
- **Boutons secondaires**: bordure or, fond transparent
- **Cards**: bg #1A1A26, border 1px solid rgba(212,175,55,0.2), shadow dorée subtile
- **Tags/Badges**: fond #2A2A3A, texte or/platine

## Categories (couleurs)
- Transport: #4A90D9 (bleu nuit)
- Hébergement: #7B68EE (violet)
- Restauration: #E8A838 (or chaud)
- Nightlife: #D4145A (rose fuchsia)
- Emploi nuit: #2ECC71 (vert émeraude)
- Location: #FF6B35 (orange luxe)
- VIP/Privé: #D4AF37 (or pur)
- Entertainment: #9B59B6 (violet riche)
